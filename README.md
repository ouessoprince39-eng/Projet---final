# Projet-Final Data Analyst - Conception d'un système intelligent d'audit et de prevention des anaomalies de paie

## Objectif 
Détecter et prévenir automatiquement les anomalies sur les fiches de paie grâce à un système intelligent, afin de réduire les erreurs, les délais de traitement et les amendes auxquelles les entreprises peuvent être exposées en cas d’irrégularités

## Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Contenu 
- Analyse_des_tables_de_faits.ipynb : Notebook
- Analyse_des_tables_de_dimensions.ipynb : Notebook
- Dashboard_tables_de_faits.pbix : Power Bi
- Dashboard_tables_de_dimensions.pbix : Power Bi
- Présentation_fictive.pdf : Présentation

  - Ouvrir le notebook avec Jupyter / Colab
  - Ouvrir le .pbix avec Power BI Desktop
